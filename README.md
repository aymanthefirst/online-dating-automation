# online-dating-automation

Tinder auto Swipe
Auto swipes tinder application using capybara, selenium. Just change the variables in login_service.rb, username and password to match your own. it swipes randomly left, or right, at random times between one and 3 seconds, for a random amount of times between 100 and 300, to match human activity and not be detected as an automator. 

Plenty of Fish
Automatically sends messages to people in a specified search. Just change the variables in login_step_defs.rb, username and password to match your own. Uses similar methods to above to randomise activity.
